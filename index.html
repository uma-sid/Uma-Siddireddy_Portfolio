Video Games Sales Analysis using Tableau & Power BI

The life time sales of different video games (scraped from Metacritic website) are analysed in this project using Tableau. The dataset consists of 16 features out of the box. Four new calculated features are generated for the analysis purpose. The whole project is driven based on the research questions framed as below.

  • Is there a correlation between the different performance metrics(eg. Critic Score, User count etc) with respect to the available dimensions(eg. Genre, platform etc)?
  • Who leads the development of the most preferred platform based games with in the last 10 years across different regions of the world?

Customer Demographics Analytics using Hadoop MapReduce

Customer Demographics of a bicycle spare parts company are analysed in this project. The data sets are stored in Hadoop Distributed File System (HDFS). This project involves joining of two different datasets, namely sales transactions and customer demographics and analysing them together. The whole project is done using Hadoop-MapReduce Jobs by designing 3 mappers and 2 reducers in contradiction to the conventional MapReduce jobs involving a single mapper and reducer. The obtained final dataset is then visualized in Tableau to answer the following research question.

  • Is the number of customers that purchasing the small size products is declining and how does this vary in terms of people having a car?
